# Plan 65-02: LightRAG Integration

## Goal
Implement LightRAG as the RAG orchestration layer connecting embeddings, vectors, and graphs.

## Components

1. **jina-embeddings-v3** — Lithuanian-optimized embeddings
2. **pgvector + pgvectorscale** — Vector storage with DiskANN
3. **LightRAG** — Orchestration layer

## Tasks

### Task 1: Embedding Service (3h)

jina-embeddings-v3 API integration, batch generation, caching.

### Task 2: Vector Storage Setup (3h)

pgvector extension, pgvectorscale DiskANN indexes, vector table schema.

### Task 3: LightRAG Configuration (4h)

LightRAG initialization, graph + vector hybrid retrieval, context window management.

### Task 4: Index Pipeline (2h)

Automatic indexing on content creation, incremental updates.

## Verification

```bash
# Embed text → store in pgvector → retrieve via similarity
# Graph-enhanced retrieval returns related entities
```
