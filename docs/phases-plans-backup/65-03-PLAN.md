# Plan 65-03: Retrieval Pipeline

## Goal
Create unified search interface combining vector similarity with graph relationships.

## Retrieval Modes

1. **Vector-only:** Pure semantic similarity
2. **Graph-only:** Relationship traversal
3. **Hybrid:** Vector candidates → graph expansion → reranking

## Tasks

### Task 1: RetrievalService (4h)

Query embedding, vector search, graph expansion, result merging and reranking.

### Task 2: Context Builder (3h)

Build LLM-ready context: prospect context, keyword relationships, related pages/topics.

### Task 3: API Endpoints (2h)

- `POST /api/rag/query` — General retrieval
- `POST /api/rag/prospect-context` — Prospect-specific
- `POST /api/rag/keyword-expansion` — Keyword graph traversal

### Task 4: Caching Layer (1h)

Cache frequent queries with TTL.
