# Plan 63-01: Business Context Extraction + Classification Foundation

## Goal
Enable contextual keyword filtering by extracting what businesses DON'T do (negative associations) and their business model (B2B/B2C/ecommerce), then build the multi-tier classification cascade infrastructure.

## Context
- **Gap G1:** No negative association extraction (can't filter adjacent verticals)
- **Gap G2:** No business model detection (can't filter by B2B/B2C intent)
- **Gap G3:** No binary relevance classification (need 0.85 confidence gate)
- **Reference:** 56-AUDIT.md, 56-CASCADE-ARCHITECTURE.md, 56-RESEARCH-GROK.md

## Tasks

### Task 1: Extend ConversationExtractor Prompt (2h)

**File:** `open-seo-main/src/server/features/prospects/prompts/conversation-extractor.xml`

Add extraction blocks for negativeAssociations (notServices, competitors, adjacentVerticals, wrongIntent) and businessModel (b2b, b2c, ecommerce, serviceProvider, manufacturer, reseller).

### Task 2: Update ExtractedProspectData Schema (1h)

**File:** `open-seo-main/src/db/prospect-schema.ts`

Add NegativeAssociationsSchema and BusinessModelSchema with backwards compatibility.

### Task 3: Create Classification Types (1.5h)

**File:** `open-seo-main/src/server/features/keywords/services/classification/types.ts`

Implement BusinessContextSchema, KeywordTypeSchema, Pass1ResultSchema, ClassifiedKeywordSchema.

### Task 4: Create Model Configuration (1h)

**File:** `open-seo-main/src/server/features/keywords/services/classification/model-config.ts`

Configure Grok 4.1 ($0.20/1M) as pass1Primary, Gemini as fallback, Claude Sonnet as pass2.

### Task 5: Create ModelSelector Service (2h)

**File:** `open-seo-main/src/server/features/keywords/services/classification/ModelSelector.ts`

Health tracking, consecutive failure threshold (3), recovery time (60s), fallback selection.

### Task 6: Create ErrorHandler Service (1.5h)

**File:** `open-seo-main/src/server/features/keywords/services/classification/ErrorHandler.ts`

Handle rate limit, timeout, parse error, server error, context too long.

### Task 7: Create Pass1 Classifier (3h)

**File:** `open-seo-main/src/server/features/keywords/services/classification/Pass1Classifier.ts`

Grok 4.1 via xAI API (OpenAI SDK compatible), batch 200 keywords, return clearly_include/clearly_exclude/uncertain.

### Task 8: Create Pass2 Classifier (2h)

**File:** `open-seo-main/src/server/features/keywords/services/classification/Pass2Classifier.ts`

Claude Sonnet for uncertain keywords, assign keyword type, flag borderline for human review.
