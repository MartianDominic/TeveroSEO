# Plan 64-02: Delta Crawling Cascade

## Goal
Implement L0→L1→L2→L3 delta crawling cascade to skip unchanged content.

## Delta Cascade Levels

| Level | Check | Exit Condition | Cost |
|-------|-------|----------------|------|
| L0 | Sitemap lastmod | timestamp unchanged | Free |
| L1 | Conditional GET | 304 Not Modified | 1 HEAD request |
| L2 | Template-aware hash | content hash match | Parse only |
| L3 | Full reprocess | always runs | Full cost |

## Tasks

### Task 1: Create DeltaCrawlService (4h)

Cascade checking through all 4 levels with early exit.

### Task 2: Create Template-Aware Hasher (2.5h)

Hash main content only, ignore nav/header/footer/dynamic elements.

### Task 3: Create CrawlState Repository (2h)

Persist lastmod, etag, hashes in PostgreSQL.

### Task 4: Integrate with CrawlService (2h)

Run delta check before singleflight, update state after crawl.

### Task 5: Add Delta Metrics (1.5h)

Track skip rates per level and estimated savings.
