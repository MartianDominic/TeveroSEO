# 1. SEO Piltuvas: TOFU/MOFU/BOFU

## Kas yra SEO piltuvas?

Ivaizduokite piltuva, per kuri liesi vandeni. Virsuje platus atidaras (daug zmoniu), apacioje siauras (tie, kas perka). SEO piltuvas veikia taip pat:

```
    ╔═══════════════════════════════════════╗
    ║           TOFU (Virsus)               ║
    ║   "Ka tai yra? Kodel man reikia?"     ║
    ║         4,400+ paieskomu/men.         ║
    ╠═══════════════════════════════════════╣
         ╔═══════════════════════════╗
         ║      MOFU (Vidurys)       ║
         ║  "Kokie variantai yra?"   ║
         ║    1,000-2,500/men.       ║
         ╠═══════════════════════════╣
              ╔═════════════════╗
              ║  BOFU (Apacia)  ║
              ║ "Noriu pirkti"  ║
              ║    30/men.      ║
              ╚═════════════════╝
```

## Trys etapai paprasta kalba

| Etapas | Pavadinimas | Kliento klausimas | Jusu atsakymas |
|--------|-------------|-------------------|----------------|
| **TOFU** | Top of Funnel (Virsus) | "Reikia kazko darbuotojams..." | Edukacinis turinys apie darbo rubus |
| **MOFU** | Middle of Funnel (Vidurys) | "Iesko konkretaus produkto..." | Produktu palyginimas, gidas |
| **BOFU** | Bottom of Funnel (Apacia) | "Noriu uzsakyti su logotipu..." | Kainu skaiciuokle, uzklausos forma |

## Realus pavyzdys: HR vadove Vilniuje

**Situacija:** Jurgita, HR vadove 50 darbuotoju logistikos imoneje, gauna uzduoti: "Reikia kazko darbuotojams ziemai."

| Savaite | Kas vyksta | Google paieska | Kokio turinio iesko |
|---------|------------|----------------|---------------------|
| 1 | Tyrineja galimybes | "darbo rubai" | Bendras straipsnis apie darbo rubu tipus |
| 2 | Supranta, ko reikia | "darbo striukes" | Striukiu palyginimas: medziagos, siltumas |
| 3 | Iesko tiekeju | "firmines striukes su logotipu" | Jusu produktu puslapis |
| 4 | Uzklausa | "striukes siuvinejimu Vilnius" | Kainos, terminai, uzklausos forma |

**Svarbu:** Jei nesate pirmame etape — Jurgita jusu nesurado ir nuzengus i antra etapa.

# 2. TOFU Hub Strategija

## Kas yra "Content Hub" (Turinio centras)?

**Analogija:** Ivaizduokite prekybos centra. Pagrindinis puslapis yra kaip centrinis holas, o kiekvienas straipsnis — specializuota parduotuve.

```
                    ┌─────────────────────────────┐
                    │   DARBO RUBAI (Hub)         │
                    │   Pagrindinis puslapis      │
                    │   Raktas: darbo rubai       │
                    │   4,400 paieskomu/men.      │
                    └──────────────┬──────────────┘
                                   │
         ┌─────────────────────────┼─────────────────────────┐
         │                         │                         │
         ▼                         ▼                         ▼
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│ Darbo striukes  │     │ Siuvinejimu     │     │ Uniformos       │
│ gidas           │     │ technologijos   │     │ imonems         │
│ 260/men.        │     │ 390/men.        │     │ 720/men.        │
└─────────────────┘     └─────────────────┘     └─────────────────┘
         │                         │                         │
         ▼                         ▼                         ▼
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│ Ziemines vs     │     │ Logotipu        │     │ Kaip isskirit   │
│ demisezonines   │     │ medziagos       │     │ darbuotojus     │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

## Kodel "Darbo rubai" turi buti Hub puslapis?

| Priezastis | Paaiskimas |
|------------|------------|
| **Didziausias srautas** | 4,400 paieskomu/men. — 10x daugiau nei kiti |
| **Platus teminis aprepimas** | Apima visus jusu produktus ir paslaugas |
| **Google "topical authority"** | Google mato, kad esate ekspertai sioje temoje |
| **Ilgalaikis SEO turtas** | Sukurtas vienakart, generuoja srautas metams |

## Hub puslapio struktura

**URL:** striukestau.lt/darbo-rubai/

**Turinys (2,500-3,500 zodziai):**

```
H1: Darbo Rubai Imonems: Pilnas Vadovas 2024

H2: Kas yra darbo rubai?
    - Apibrezimai
    - Tipai (striukes, liemenes, softshell)
    
H2: Darbo rubu pasirinkimo kriterijai
    - Sezonas (ziema/vasara)
    - Darbo specifika
    - Biudzetas
    
H2: Personalizavimas: logotipai ir siuvinejimas
    [Nuoroda i siuvinejimo puslapi]
    
H2: Darbo striukes: detali apzvalga
    [Nuoroda i striukiu kategorija]
    
H2: Kodel imoniu logotipas ant rubu svarbus
    - Komandos identitetas
    - Profesionalus ivaizdis
    
H2: DUK apie darbo rubus
    FAQ schema markup
```

## Vidiniai linkai (Internal Linking)

```
Hub puslapis "Darbo rubai"
     │
     ├── → /darbo-striukes/ (kategorija)
     │        ├── → /ziemines-darbo-striukes/
     │        └── → /softshell-striukes/
     │
     ├── → /siuvinejimas/ (paslauga)
     │        └── → /logotipu-technologijos/
     │
     └── → /uniformos-imonems/ (kategorija)
              └── → /sektoriu-gidas/
```

**Taisykle:** Kiekvienas puslapis turi tureti nuoroda atgal i Hub.

# 5. Kodel SEO ≠ Tiesioginis Pardavimas B2B

## Problema: BOFU Apimtis

Pasižiūrėkime i skaičius:

| Raktažodis | Paieškos/mėn. | Etapas |
|------------|---------------|--------|
| darbo rūbai | 4,400 | TOFU |
| uniformos | 720 | TOFU |
| vyriškos striukės | 1,300 | MOFU |
| žieminės striukės | 1,000 | MOFU |
| darbo striukės | 260 | MOFU |
| **drabužiai didmena** | **30** | **BOFU** |
| firmines striukes uzsakyti | ~10 | BOFU |

**Išvada:** BOFU raktažodžiai turi 100-400x mažesnę paieškos apimtį nei TOFU.

## Kodėl Taip Yra?

### 1. B2B Pirkimo Specifika

```
B2C (Vartotojas)              B2B (Įmonė)
────────────────              ────────────
• Vienas žmogus sprendžia     • 2-5 žmonės dalyvauja
• Sprendimas per 1-7 dienas   • Sprendimas per 2-8 savaites
• Ieško Google visą laiką     • Ieško Google pradžioje
• Perka online               • Perka per užklausą/sutartį
```

### 2. Kur Vyksta B2B Pardavimas?

| Kanalas | Vaidmuo | Konversijos Tikimybė |
|---------|---------|---------------------|
| Google Paieška | Atradimas, tyrinėjimas | 1-3% |
| Jūsų Svetainė | Edukacija, pasitikėjimas | 5-10% |
| El. paštas | Nurturing, follow-up | 10-15% |
| Telefono skambutis | Pardavimo uždarymas | 20-30% |
| LinkedIn | Ryšių kūrimas, outreach | 5-10% |
| Rekomendacijos | Aukščiausia konversija | 30-50% |

### 3. SEO Vaidmuo B2B Piltuvo

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   SEO DARO:                    SEO NEDARO:                     │
│   ──────────                   ─────────────                   │
│   ✓ Pritraukia naujus          ✗ Neuždaro sandorių            │
│     lankytojus                                                  │
│                                                                 │
│   ✓ Kuria prekės ženklo        ✗ Nepakeičia pardavėjo         │
│     žinomumą                                                    │
│                                                                 │
│   ✓ Edukuoja potencialius      ✗ Nekonvertuoja be             │
│     klientus                      follow-up                    │
│                                                                 │
│   ✓ Generuoja lead'us          ✗ Negeneruoja pajamų           │
│     (el. paštus, užklausas)       be pardavimų proceso        │
│                                                                 │
│   ✓ Kuria ilgalaikį turtą      ✗ Neduoda rezultatų            │
│     (turinį)                      per pirmą mėnesį            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Ką Tai Reiškia Jūsų Strategijai?

### SEO = Piltovo Viršus

SEO užpildo piltuvą potencialiais klientais. Jūsų darbas — juos konvertuoti per kitus kanalus.

### Papildomi Kanalai

| Kanalas | Kam Naudoti | Investicija |
|---------|-------------|-------------|
| **LinkedIn** | HR vadovų outreach, content | 5-10 val/sav |
| **El. pašto sekos** | Lead nurturing | Automatizuota |
| **Skambučiai** | Užklausų follow-up | Per 24h |
| **Google Ads** | BOFU raktažodžiai | 200-500 EUR/mėn |
| **Rekomendacijos** | Programa esamiems klientams | Nuolaidos |

### Realistiški Lūkesčiai

| Metrika | 3 mėn. | 6 mėn. | 12 mėn. |
|---------|--------|--------|---------|
| Organinis srautas | +50% | +150% | +300% |
| Užklausos iš SEO | 5-10/mėn | 15-25/mėn | 30-50/mėn |
| Sandoriai iš SEO | 1-2/mėn | 3-5/mėn | 8-12/mėn |
| ROI | Neigiamas | Break-even | 3-5x |

**Svarbu:** SEO yra maratonas, ne sprintas. Rezultatai ateina po 6-12 mėn.
