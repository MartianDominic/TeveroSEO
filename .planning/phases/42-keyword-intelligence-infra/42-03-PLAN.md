<objective>
Implement unified embedding service using jina-embeddings-v3 with Matryoshka truncation to 384-dim for storage efficiency.

Purpose: jina-v3 is the best embedding model for Lithuanian language quality (Cohen's kappa 0.62, AUC-ROC 0.887 on LtHate benchmark per ADR-002). Matryoshka truncation allows native dimension slicing to 384-dim without PCA fitting.

Output: Embedding service with Jina API integration, PostgreSQL schema with pgvector + DiskANN, and query helpers.
</objective>
