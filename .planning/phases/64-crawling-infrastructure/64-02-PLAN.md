```typescript
export interface DeltaResult {
  action: "skip" | "fetch" | "process";
  reason: string;
  layer: "L0" | "L1" | "L2" | "L3";
}

export async function deltaCascade(
  url: string,
  tenantId: string,
  sitemapInfo: SitemapUrl | null,
  lastCrawledAt: Date | null,
  cachedHeaders: CachedHeaders | null,
  deltaService: DeltaSyncService
): Promise<DeltaResult>;
```

Layer logic:
- L0: If sitemapInfo?.lastmod && lastCrawledAt, use filterByLastmod. Skip if unchanged.
- L1: If cachedHeaders?.etag || cachedHeaders?.lastModified, use conditionalGet. Skip on 304.
- L2: If L1 returns changed with HTML, check via DeltaSyncService. Skip if UNCHANGED.
- L3: Default fallback for new URLs or when all layers indicate change.

Shopify handling (per RESEARCH.md pitfall 2): Treat L0 as negative-only signal. Changed lastmod -> verify with L1/L2.

Update index.ts to export delta-cascade and conditional-get modules.
