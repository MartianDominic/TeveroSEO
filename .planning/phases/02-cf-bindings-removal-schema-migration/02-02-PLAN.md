<objective>
Migrate every Drizzle table definition from SQLite to PostgreSQL. Swap `sqliteTable` for `pgTable`, convert boolean integer columns to native `boolean()`, convert JSON text columns to `jsonb()`, and convert every timestamp column to `timestamp({ withTimezone: true, mode: "date" })`.

Purpose: Without a PG-dialect schema, `drizzle-kit generate` cannot emit PostgreSQL migrations and the node-postgres driver will throw type errors. This is the canonical data model for the entire app.
Output: Two schema files using `drizzle-orm/pg-core` with column types correct for PostgreSQL.
</objective>
