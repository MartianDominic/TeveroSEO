## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| URL search params → component | Search params are strings from URL; normalize and validate before use |
| component → server action | Server action validates input before forwarding to open-seo |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-08 | Input Validation | keyword/domain/target in search params | mitigate | Components pass raw values to server actions; server actions forward to open-seo which validates. Apply string length guard (max 500 chars) in server actions. |
| T-10-09 | Information Disclosure | DataForSEO error messages exposed via server action throw | mitigate | Server actions catch upstream errors; return user-friendly message. Do not forward raw DataForSEO API error bodies. |
