Create types.ts with:
```typescript
export interface DetailedLink {
  targetUrl: string;
  targetPageId: string | null;
  anchorText: string;
  context: string;
  position: 'body' | 'sidebar' | 'footer' | 'nav' | 'header';
  paragraphIndex: number | null;
  isDoFollow: boolean;
  linkType: 'contextual' | 'nav' | 'footer' | 'sidebar' | 'image';
}
```

Create link-extractor.ts with:
1. extractDetailedLinks(html, pageUrl, urlToPage) - main extraction function
   - Use cheerio for HTML parsing
   - Filter out external links (different origin)
   - Filter out javascript:, mailto:, tel:, # links
   - Resolve relative URLs using URL constructor
   - Call classifyLinkPosition for each link
   - Call getParagraphIndex for body links
   - Extract surrounding context (~50 chars)
   - Check rel attribute for nofollow

2. classifyLinkPosition($link, $) - position classification
   - Check parent elements for nav patterns (nav tag, navigation/menu class)
   - Check for header patterns (header tag, header class)
   - Check for footer patterns (footer tag, footer class)
   - Check for sidebar patterns (aside tag, sidebar/widget class)
   - Default to 'body' for main content

3. getParagraphIndex($link, $) - paragraph index
   - Find containing paragraph
   - Find main content container (main, article, .content)
   - Count paragraphs to determine index
   - Return null if not in paragraph

4. extractContext($link, $) - surrounding text
   - Get parent text content
   - Find link text position
   - Extract ~25 chars before and after
  </action>

  <action>
Create graph-builder.ts with:

1. buildLinkGraph(params: { auditId, clientId }) - main entry point
   - Fetch all audit_pages for the audit
   - Build URL -> pageId lookup map
   - For each page:
     - Fetch cached HTML (from audit storage or re-fetch)
     - Call extractDetailedLinks
     - Insert each link into link_graph with onConflictDoNothing
   - Call computePageLinkMetrics
   - Call detectOrphanPages

2. computePageLinkMetrics(auditId, clientId)
   - Aggregate inbound counts per targetUrl from link_graph
   - Count by position (body, nav, footer, sidebar)
   - Count by type (exactMatch, branded, doFollow)
   - Count first paragraph links
   - Aggregate outbound counts per sourceUrl
   - Count internal vs external
   - Calculate unique anchors and top anchors (jsonb)
   - Calculate anchor distribution percentages
   - Upsert into page_links table

3. detectOrphanPages(auditId, clientId)
   - Query pages with zero rows in link_graph where page is target
   - Cross-reference with sitemap discovery
   - Insert into orphan_pages with status='detected'
   - Set discovery source based on how page was found

4. Helper: getAuditPageHtml(pageUrl, auditId) - get cached HTML
   - Check if HTML cached in audit storage
   - If not cached, return null (will need re-crawl)

Export all from index.ts barrel.
