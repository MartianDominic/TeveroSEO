## Decisions Made
- Used RGB color format (`rgb(59, 130, 246)`) instead of hex or CSS variables per RESEARCH.md pitfall #2 (Puppeteer SVG hex color bug)
- Set explicit chart dimensions (700x280) instead of ResponsiveContainer per RESEARCH.md pitfall #3 (legend off-screen in PDF)
- Used Intl.DateTimeFormat/NumberFormat for locale-aware formatting (no external i18n dependency for initial implementation)
- Chart components are client-side (`"use client"`) due to Recharts; other sections are server components
