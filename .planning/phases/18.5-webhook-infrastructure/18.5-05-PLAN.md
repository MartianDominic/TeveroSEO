# Plan 18.5-05: Webhook Configuration UI

## Objective
Create webhook management UI for clients to configure their webhooks.

## Tasks

### Task 1: Server Actions

Create `apps/web/src/actions/webhooks.ts`:
- listWebhooks, createWebhook, updateWebhook, deleteWebhook
- getWebhookDeliveries, getEventRegistry

### Task 2: Webhook Components

- WebhookList - Table of configured webhooks
- WebhookForm - Create/edit webhook form
- WebhookDeliveries - Delivery history viewer
- EventSelector - Multi-select for events

### Task 3: Webhook Settings Page

Create `/clients/[clientId]/settings/webhooks/page.tsx`
