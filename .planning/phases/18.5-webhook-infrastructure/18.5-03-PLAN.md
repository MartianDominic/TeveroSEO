# Plan 18.5-03: HMAC Signing + Delivery Worker

## Objective
Create webhook delivery worker with HMAC signature generation and retry logic.

## Tasks

### Task 1: HMAC Signing Utility

Add HMAC-SHA256 signature generation to create `X-Webhook-Signature` header.

### Task 2: Webhook Processor (webhook-processor.ts)

Create sandboxed processor:
- Fetch delivery record
- Generate HMAC signature
- POST to webhook URL
- Handle response (2xx = success, else retry)
- Update delivery status

### Task 3: Webhook Worker (webhook-worker.ts)

Standard BullMQ worker lifecycle with graceful shutdown.

## Self-Check

- [ ] HMAC-SHA256 signature in X-Webhook-Signature header
- [ ] Timestamp in X-Webhook-Timestamp header
- [ ] Delivery status updated after each attempt
- [ ] Success on 2xx, retry on other codes
- [ ] DLQ handling for exhausted retries
- [ ] TypeScript compiles clean
