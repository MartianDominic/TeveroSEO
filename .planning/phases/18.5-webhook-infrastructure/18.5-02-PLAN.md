# Plan 18.5-02: Webhook Dispatcher Service

## Objective
Create dispatcher that takes events, finds matching webhooks, creates delivery records, and enqueues delivery jobs.

## Tasks

### Task 1: Webhook Service (webhooks.ts)

Create `open-seo-main/src/services/webhooks.ts`:

- `createWebhook()` - Create webhook with secret generation
- `updateWebhook()` - Update webhook config
- `deleteWebhook()` - Delete webhook
- `getWebhooksByScope()` - List webhooks for scope
- `findMatchingWebhooks()` - Find webhooks matching event type and scope

### Task 2: Webhook Dispatcher (webhook-dispatcher.ts)

Create `open-seo-main/src/services/webhook-dispatcher.ts`:

- `emitEvent()` - Main dispatch function
  1. Build full payload with idempotency key
  2. Find matching webhooks (client > workspace > global)
  3. Create delivery records
  4. Enqueue delivery jobs

### Task 3: Generate ID Utilities

Add `generateEventId()` and `generateIdempotencyKey()` functions.
