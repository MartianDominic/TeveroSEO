```css
/* Card shadows */
--shadow-card: 0 1px 2px rgba(20,20,26,0.04), 0 2px 4px rgba(20,20,26,0.02), 0 0 0 1px var(--hairline-3);
--shadow-lift: 0 2px 8px rgba(20,20,26,0.08), 0 4px 16px rgba(20,20,26,0.04), 0 0 0 1px var(--hairline-2);

/* Card hover transition */
transition: box-shadow var(--motion-hover), transform var(--motion-hover);
transform: translateY(-1px); /* on hover */

/* Semantic colors for badges */
--success: #1B6E45;  --success-soft: #EAF2EE;  /* low competition */
--warning: #A87F1A;  --warning-soft: #F4EDDA;  /* medium competition */
--error: #9B2C2C;    --error-soft: #F4E6E6;    /* high competition */

/* Skeleton shimmer (instead of animate-pulse) */
.skeleton { animation: skeleton-shimmer 1.5s var(--ease-smooth) infinite; }
```

Tailwind utility mappings:
```
shadow-card                -> card shadow at rest
shadow-lift                -> card shadow on hover
hover:shadow-lift          -> hover state shadow
hover:-translate-y-px      -> lift effect
transition-all duration-[280ms] ease-[cubic-bezier(0.16,1,0.3,1)] -> motion-hover

bg-success-soft text-success  -> low competition badge
bg-warning-soft text-warning  -> medium competition badge
bg-error-soft text-error      -> high competition badge

skeleton                   -> loading shimmer (defined in globals.css)
```
