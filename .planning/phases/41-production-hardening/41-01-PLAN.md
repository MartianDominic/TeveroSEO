### Task 3: Fix InMemoryFindingsRepository factory
**File:** `apps/web/src/lib/audit/repositories/FindingsRepository.ts`
**Lines:** 264-275 (factory function)

**Current (problematic):**
```typescript
export function createFindingsRepository(): FindingsRepository {
  // Could return InMemoryFindingsRepository if condition wrong

```

**Fix:**
```typescript
export function createFindingsRepository(): FindingsRepository {
  if (process.env.NODE_ENV === 'test') {
    return new InMemoryFindingsRepository();

}
```

### Task 6: Add startup validation for dangerous flags
**File:** `AI-Writer/backend/main.py` (or config validation)

**Add validation:**
```python
import os

def validate_production_config():
    """Reject dangerous configurations in production."""
    if os.getenv("ENV", "development") == "production":
        if os.getenv("DISABLE_AUTH", "false").lower() == "true":
            raise ValueError("DISABLE_AUTH=true is not allowed in production")
        if os.getenv("QUALITY_GATE_ENABLED", "true").lower() == "false":
            raise ValueError("QUALITY_GATE_ENABLED=false is not allowed in production")

# Call at startup
validate_production_config()
```

**Verification:**
```bash
grep -n "validate_production_config\|DISABLE_AUTH.*production" AI-Writer/backend/main.py
# Should show validation function
```
