<objective>
Implement pSEO (programmatic SEO) pattern detection for Lithuanian keywords.

Purpose: Detect when multiple keywords follow a "[service] [CITY]" pattern, indicating a template opportunity. Instead of creating 50 separate pages for "plovykla vilniuje", "plovykla kaune", etc., the system should identify this as ONE pSEO template opportunity with estimated 50+ pages.

Output: PSEODetector service that groups keywords by city-stripped pattern, scores opportunities, and generates URL template recommendations.
</objective>

  <action>
Implement PSEODetector per algorithm in CONTEXT.md:

1. **detectPSEOPatterns(keywords, config)** - Main detection function:
   - Input: ClassifiedKeyword[] with volume/difficulty, PSEODetectorConfig
   - For each keyword, extract city using extractCityFromKeyword
   - Group keywords by "keyword without city" (the pattern base)
   - Filter to groups with minClusterSize (default 3) or more keywords
   - For each qualifying group, create PSEOCluster:
     - pattern: `${base} [CITY]`
     - template: generateURLTemplate(base) -> slugify, prepend /
     - keywords: all matching keyword strings
     - cities: unique extracted cities
     - estimatedPages: LITHUANIAN_CITIES.length (50+)
     - totalVolume: sum of volumes
     - avgDifficulty: average of difficulties
     - opportunityScore: computePSEOScore()

2. **computePSEOScore(cluster)** - Scoring per CONTEXT.md formula:
   - volumeScore: min(totalVolume / 5000, 1.0) * 0.35
   - cityScore: min(cityCount / 10, 1.0) * 0.25
   - difficultyScore: (1 - avgDifficulty/100) * 0.20
   - funnelScore: average funnel value (BOFU=1.0, MOFU=0.7, TOFU=0.4) * 0.20
   - Return weighted sum

3. **generateURLTemplate(base)** - URL slug generation:
   - Lowercase, replace spaces with hyphens
   - Remove diacritics (lithuanianToAscii)
   - Prepend "/" and append "/{city}"
   - Example: "automobiliu plovykla" -> "/automobiliu-plovykla/{city}"

4. **PSEODetector class** - Stateful detector with config:
   - Constructor takes PSEODetectorConfig
   - detect(keywords) method returns PSEOCluster[]
   - Sorts clusters by opportunityScore descending
  </action>
