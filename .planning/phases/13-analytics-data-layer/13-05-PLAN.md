```typescript
if (process.env.WORKER_MODE === "true" || process.env.NODE_ENV === "development") {

```

In the shutdown handler:
```typescript
process.on("SIGTERM", async () => {
  console.log("[server] SIGTERM received, shutting down...");
  await stopAuditWorker();
  await stopAnalyticsWorker();
  await closeRedis();
  process.exit(0);
});

process.on("SIGINT", async () => {
  console.log("[server] SIGINT received, shutting down...");
  await stopAuditWorker();
  await stopAnalyticsWorker();
  await closeRedis();
  process.exit(0);
});
```

  <action>
Add the new environment variables required for the analytics system.

Required new variables:
- INTERNAL_API_KEY: Shared secret for service-to-service auth
- GOOGLE_CLIENT_ID: Google OAuth client ID (for token refresh)
- GOOGLE_CLIENT_SECRET: Google OAuth client secret (for token refresh)
- AIWRITER_INTERNAL_URL: URL of AI-Writer internal API (for open-seo-worker)
- OPEN_SEO_WORKER_URL: URL of open-seo-worker internal API (for AI-Writer)

Add to open-seo and open-seo-worker services:
```yaml
services:
  open-seo:
    environment:
      # ... existing vars ...
      - INTERNAL_API_KEY=${INTERNAL_API_KEY}
      - AIWRITER_INTERNAL_URL=http://ai-writer-backend:8000
      - GOOGLE_CLIENT_ID=${GOOGLE_CLIENT_ID}
      - GOOGLE_CLIENT_SECRET=${GOOGLE_CLIENT_SECRET}

  open-seo-worker:
    environment:
      # ... existing vars ...
      - INTERNAL_API_KEY=${INTERNAL_API_KEY}
      - AIWRITER_INTERNAL_URL=http://ai-writer-backend:8000
      - ALWRITY_DATABASE_URL=${ALWRITY_DATABASE_URL}
      - GOOGLE_CLIENT_ID=${GOOGLE_CLIENT_ID}
      - GOOGLE_CLIENT_SECRET=${GOOGLE_CLIENT_SECRET}

  ai-writer-backend:
    environment:
      # ... existing vars ...
      - INTERNAL_API_KEY=${INTERNAL_API_KEY}
      - OPEN_SEO_WORKER_URL=http://open-seo-worker:3001
```

Note: GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET should already be in ai-writer-backend from Phase 12 OAuth setup. Verify and add if missing.
