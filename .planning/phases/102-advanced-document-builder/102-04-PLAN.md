Create analytics-service.ts with:

1. recordBlockView(blockId: string, variantId?: string):
   - Redis INCR on key per D-04 pattern
   - Also add to time-series sorted set for decay analysis

2. recordBlockDwell(blockId: string, variantId: string | undefined, dwellMs: number):
   - Redis INCRBY for cumulative dwell time
   - Track in separate key: block:{blockId}:dwell

3. getBlockAnalytics(blockId: string):
   - Get impressions, conversions, avg dwell from Redis
   - Return BlockAnalytics interface

4. calculateCorrelation(blockId: string, variantId?: string):
   - Query proposals with this block that have outcome (won/lost)
   - Calculate correlation coefficient
   - Return { correlation: number, wonCount: number, lostCount: number, confidence: number }

5. markConversion(proposalId: string, outcome: 'won' | 'lost'):
   - Find all blocks in proposal
   - Increment conversion counters for each variant shown

Redis key patterns per D-04:
- `block:{blockId}:views` - total views
- `block:{blockId}:variant:{variantId}:views` - variant views
- `block:{blockId}:dwell` - cumulative dwell time
- `block:{blockId}:views:ts` - time-series sorted set
