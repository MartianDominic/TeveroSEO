Create document-builder.ts with Drizzle schema:

1. persuasionBlocks table:
   - id (text PK), proposalId (text FK), workspaceId (text)
   - type (text - PersuasionBlockType), position (integer)
   - content (jsonb - TipTap), styling (jsonb)
   - persuasionMeta (jsonb - aiHints, frameworkId, isRequired)
   - viewCount (integer default 0), dwellTimeMs (integer default 0)
   - createdAt, updatedAt timestamps
   - Indexes on proposalId, workspaceId

2. blockVariants table per D-02 (normalized, not JSONB array):
   - id (text PK), parentBlockId (text FK to persuasionBlocks ON DELETE CASCADE)
   - variantName (text - "Control", "Variant A", etc.)
   - content (jsonb), styling (jsonb nullable)
   - weight (integer default 50, CHECK 0-100)
   - impressions (integer default 0), conversions (integer default 0)
   - status (text - 'active' | 'paused' | 'winner' | 'loser')
   - createdAt timestamp
   - Indexes on parentBlockId, status

3. proposalStructures table:
   - id (text PK), proposalId (text), workspaceId (text)
   - frameworkId (text nullable), frameworkName (text nullable)
   - blockOrder (jsonb - array of block IDs)
   - validationResult (jsonb nullable)
   - createdAt, updatedAt timestamps
