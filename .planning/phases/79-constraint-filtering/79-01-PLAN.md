3. **checkGeoFilter** function:
   - Input: keyword string, geoClassification object, GeoConstraints
   - Return early `{ passes: true }` if no constraints
   - If geoClassification.passesGeoFilter === false, return `{ passes: false, reason: 'geo:wrong_city:${city}' }`
   - If generic (no city) and !genericAllowed, return `{ passes: false, reason: 'geo:generic_not_allowed' }`
   - If city not in includeCities (when includeCities provided), return `{ passes: false, reason: 'geo:not_in_target_list' }`
   - Otherwise `{ passes: true }`

4. **checkNegativeFilter** function:
   - Input: keyword string, NegativeFilters
   - Return early if no filters
   - Check excludeTerms (case-insensitive substring match)
   - Check excludeBrands (case-insensitive)
   - Check excludeIntents (case-insensitive)
   - Check DEFAULT_NEGATIVE_PATTERNS if defaultPatterns=true
   - Return first match as `{ passes: false, reason: 'negative:term:${matched}' }`

5. **checkAudienceFilter** function:
   - Input: keyword string, AudienceConstraints
   - Return early if no constraints
   - If b2bOnly, check B2C_PATTERNS - if match, return `{ passes: false, reason: 'audience:b2c_excluded' }`
   - Otherwise `{ passes: true }`

6. **checkRelevanceFilter** function:
   - Input: combinedScore number | undefined, threshold number
   - If combinedScore undefined, return `{ passes: true }` (no data = pass)
   - If combinedScore < threshold, return `{ passes: false, reason: 'relevance:below_threshold:${score.toFixed(2)}' }`
   - Otherwise `{ passes: true }`
