**platform-connection-schema.ts:**
```typescript
export const OAUTH_PLATFORM_TYPES = [
  'google_search_console', 'google_analytics', 'google_business_profile',
  'wordpress_com', 'wordpress_org', 'shopify', 'wix', 'squarespace', 
  'webflow', 'hubspot', 'bigcommerce', 'magento', 'drupal', 'ghost', 'bing_webmaster'
] as const;

export const OAUTH_CONNECTION_STATUS = [
  'pending', 'connecting', 'active', 'expired', 'revoked', 'error'
] as const;

export const CREDENTIAL_TYPES = ['oauth', 'app_password', 'api_key'] as const;
export const SYNC_SCHEDULES = ['hourly', 'daily', 'weekly', 'manual'] as const;
```

Columns per DESIGN.md:
- id: text primary key
- workspaceId: text not null
- prospectId: text (optional, nullable for workspace-level like Google)
- platform: text not null (from OAUTH_PLATFORM_TYPES)
- platformAccountId, platformAccountName, platformSiteUrl: text nullable
- accessTokenEncrypted, refreshTokenEncrypted: text nullable
- tokenExpiresAt: timestamp nullable
- tokenType: text default 'Bearer'
- credentialType: text nullable (from CREDENTIAL_TYPES)
- credentialsEncrypted: text nullable (for non-OAuth like WP app passwords)
- status: text not null default 'pending' (from OAUTH_CONNECTION_STATUS)
- lastSyncAt, lastSyncStatus, lastError, syncSchedule
- scopesRequested, scopesGranted: jsonb
- connectedAt, connectedBy, revokedAt, revokedBy
- createdAt, updatedAt: timestamp with timezone

Indexes:
- idx_platform_connections_workspace_prospect ON (workspaceId, prospectId)
- idx_platform_connections_status ON (status)
- idx_platform_connections_expiry ON (tokenExpiresAt) WHERE status = 'active'

**oauth-state-schema.ts:**
- id: text primary key
- state: text unique not null (random CSRF token)
- platform, workspaceId, prospectId (nullable), userId: text
- redirectUri: text
- scopes: jsonb
- expiresAt: timestamp not null (10 minutes from creation)
- usedAt: timestamp nullable
- createdAt: timestamp

**platform-data-cache-schema.ts:**
- id: text primary key
- connectionId: text references platformConnections.id
- dataType: text not null (e.g. 'search_queries', 'pages')
- dateRange: text nullable
- data: jsonb
- fetchedAt, expiresAt: timestamp

**OAuthProviderBase.ts:**
```typescript
export interface OAuthProvider {
  readonly name: string;
  readonly platform: OAuthPlatform;
  
  getAuthorizationUrl(state: string, options?: { scopes?: string[] }): string;
  exchangeCodeForTokens(code: string): Promise<TokenSet>;
  refreshAccessToken(refreshToken: string): Promise<TokenSet>;
  revokeToken(token: string): Promise<void>;
}

export abstract class OAuthProviderBase implements OAuthProvider {
  abstract readonly name: string;
  abstract readonly platform: OAuthPlatform;
  protected abstract readonly config: OAuthConfig;
  
  abstract getAuthorizationUrl(state: string, options?: { scopes?: string[] }): string;
  abstract exchangeCodeForTokens(code: string): Promise<TokenSet>;
  abstract refreshAccessToken(refreshToken: string): Promise<TokenSet>;
  abstract revokeToken(token: string): Promise<void>;
}
```
