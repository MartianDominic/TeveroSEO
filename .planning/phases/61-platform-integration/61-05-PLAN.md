Create UniversalCrawler per D-16 through D-19:

**UniversalCrawler.ts:**

Interfaces:
- CrawlOptions: { maxPages?, respectRobots?, renderJs?: boolean | 'auto', timeout?, userAgent? }
- PageData: { title, metaDescription?, h1[], h2[], canonicalUrl?, ogTitle?, ogDescription?, internalLinks[] }
- CrawlResult: { status: 'success'|'blocked'|'error', method?: 'fetch'|'playwright', data?, sitemapUrls?, reason?, error? }

DEFAULT_OPTIONS:
- maxPages: 100
- respectRobots: true
- renderJs: 'auto'
- timeout: 30000
- userAgent: 'TeveroSEO-Bot/1.0 (+https://tevero.io/bot)'

Class UniversalCrawler:
- Private browser: Browser | null

Method crawl(url, options): Promise<CrawlResult>
1. Check robots.txt (D-16) - if respectRobots true, fetch and check isAllowed
2. Try sitemap discovery (D-17) - find and parse sitemap
3. Detect JS rendering need (D-18) - if 'auto', use SPADetector.checkUrl
4. Crawl with appropriate method (D-19):
   - If needsJs: crawlWithPlaywright
   - Else: crawlWithFetch

Method crawlWithFetch(url, options): Promise<PageData>
- Use fetch with timeout and user agent
- Extract page data from HTML with regex

Method crawlWithPlaywright(url, options): Promise<PageData>
- Launch chromium headless if not already running
- Navigate with waitUntil: 'networkidle'
- Wait for h1/article/main selector (10s timeout, catch errors)
- Extract data with page.evaluate

Method extractPageData(html, url): PageData
- Parse title, meta description, h1[], h2[], canonical, og:title, og:description, internal links
- Use regex extraction

Method close(): Promise<void>
- Close browser instance

**API routes:**

apps/web/src/app/api/crawl/route.ts:
- POST: Accept { url }, validate, create UniversalCrawler, crawl, return result
- Use Clerk auth

apps/web/src/app/api/crawl/[jobId]/route.ts:
- GET: Return cached crawl result by jobId (for async implementation later)

Create crawler/index.ts barrel export for all crawler modules.
