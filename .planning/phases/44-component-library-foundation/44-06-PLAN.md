1. Create packages/ui/src/components/loading-skeleton.tsx:

LoadingSkeletonProps interface:
```typescript
interface LoadingSkeletonProps {
  variant: 'text' | 'card' | 'table' | 'chart' | 'avatar' | 'button';
  lines?: number;
  rows?: number;
  width?: string | number;
  height?: string | number;
  className?: string;
}
```

Implementation:

CRITICAL: Never use animate-pulse. Use CSS opacity animation:
```css
@keyframes skeleton-shimmer {
  0% { opacity: 0.5; }
  50% { opacity: 1; }
  100% { opacity: 0.5; }
}

.skeleton {
  background: var(--surface-3);
  animation: skeleton-shimmer 1.5s var(--ease-smooth) infinite;
}

@media (prefers-reduced-motion: reduce) {
  .skeleton {
    animation: none;
    opacity: 0.7;
  }
}
```

Add the keyframes to the component using a <style> tag or ensure it's in globals.css.

Variants:
- text: Single or multiple lines with varying widths (90%, 75%, 60%)
- card: Full card placeholder (16:9 aspect ratio by default)
- table: Row placeholders (default 3 rows)
- chart: Rectangle with chart proportions
- avatar: Circle placeholder (default 40px)
- button: Rounded rectangle (default 80x36)

Base styles:
- bg-surface-3
- radius-input for text, radius-card for card, rounded-full for avatar

Implementation:

- Default emptyComponent: <EmptyState title="No data" />
- isEmpty default: Array.isArray(data) ? data.length === 0 : !data

Generic typing to support any data shape.
