## Color

**Source:** design-system-v6.md Section 2.1

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FAFAF7 (canvas) | Page background |
| Secondary (30%) | #FFFFFF (surface) | Cards, panels |
| Accent (10%) | #0F4F3D (emerald) | See reserved list below |
| Destructive | #9B2C2C | Destructive actions only |

**Full color token set:**

```css
/* Canvas / surface ladder */
--canvas:      #FAFAF7   /* page background */
--canvas-dim:  #F5F4EE   /* subtle inset surfaces */
--surface:     #FFFFFF   /* cards */
--surface-2:   #F8F8F3   /* hover state, tinted insets */
--surface-3:   #F2F1EB   /* progress bar track, skeleton */

/* Hairlines (transparent) */
--hairline:    rgba(20, 20, 26, 0.06)
--hairline-2:  rgba(20, 20, 26, 0.04)
--hairline-3:  rgba(20, 20, 26, 0.025)

/* Text ramp (warm-shifted, NOT slate) */
--text-1:      #14141A   /* headings, hero numerals */
--text-2:      #54545A   /* body */
--text-3:      #93939A   /* labels, eyebrows */
--text-4:      #C4C3BB   /* dividers, whisper text */

/* Accent (emerald) */
--accent:      #0F4F3D
--accent-2:    #1A6E55
--accent-soft: #EAF1ED
--accent-ink:  #093528
--accent-line: #C8DDD4
--accent-tint: #5F9181

/* Semantic */
--success:      #1B6E45   --success-soft: #EAF2EE
--error:        #9B2C2C   --error-soft:   #F4E6E6
--warning:      #A87F1A   --warning-soft: #F4EDDA
--info:         #2D5A87   --info-soft:    #EFF4F9
```

**Accent reserved for:**
1. Primary CTA buttons (gradient: #1A6E55 to #0F4F3D)
2. Active tab underline (2px)
3. Priority row left border (2px)
4. Progress bar fill (goal achieved)
5. Status pill for "on track"
6. Link hover state
7. Focus ring outline
8. Selected item background (accent-soft)
9. Hover-reveal arrows

**Color anti-patterns (FORBIDDEN):**
- No multiple chromatic accents on a page
- No Tailwind slate ladder (too cool)
- No #000000 or pure black text
- No tinted card backgrounds (cream is canvas only)

## Shadow System

**Source:** design-system-v6.md Section 2.9

Cards use layered ghost-edge shadows, NEVER `border: 1px solid`:

```css
/* At rest */
--shadow-card:
  0 0 0 1px rgba(20, 20, 26, 0.045),
  0 1px 2px rgba(20, 20, 26, 0.03),
  inset 0 1px 0 rgba(255, 255, 255, 0.5);

/* On hover */
--shadow-lift:
  0 0 0 1px rgba(20, 20, 26, 0.06),
  0 6px 16px -4px rgba(20, 20, 26, 0.06),
  0 16px 40px -16px rgba(20, 20, 26, 0.10),
  inset 0 1px 0 rgba(255, 255, 255, 0.55);

/* Pop (small elements) */
--shadow-pop:
  0 0 0 1px rgba(20, 20, 26, 0.05),
  0 2px 4px rgba(20, 20, 26, 0.04),
  0 12px 28px -8px rgba(20, 20, 26, 0.08);

/* Primary CTA */
--shadow-cta:
  inset 0 0 0 1px rgba(15, 79, 61, 0.6),
  inset 0 1px 0 rgba(255, 255, 255, 0.18),
  0 1px 2px rgba(15, 79, 61, 0.18),
  0 4px 12px -2px rgba(15, 79, 61, 0.20);
```

## Motion

**Source:** design-system-v6.md Section 2.10

| Token | Value | Usage |
|-------|-------|-------|
| ease-smooth | cubic-bezier(0.16, 1, 0.3, 1) | Shadow, transform |
| ease-quick | cubic-bezier(0.4, 0, 0.2, 1) | Color, background |
| motion-fast | 160ms ease-quick | Row hover |
| motion-hover | 280ms ease-smooth | Card lifts |
| motion-reveal | 240ms ease-smooth | Fade-ins |

**Motion anti-patterns:**
- No `linear` easing
- No animation > 320ms
- No `transition: all` (specify properties)
- No `transform: scale()` on hover (use translateY)
- Honor `prefers-reduced-motion`

## Design System Anti-Patterns

**Source:** design-system-v6.md Sections 15-16

Components MUST avoid these tells:

| Anti-Pattern | Correct Approach |
|--------------|-----------------|
| Inter / SF font | Geist + Newsreader |
| Tailwind slate ladder | Warm-shifted text ramp |
| `text-transform: uppercase` | `font-variant-caps: all-small-caps` |
| `border: 1px solid` on cards | Shadow system |
| `transition: all` | Specify properties |
| `transform: scale()` hover | `translateY(-1px)` |
| Animation > 320ms | Keep under 320ms |
| Multiple chromatic accents | One accent (emerald) |
| Dot-grain backgrounds | Plain canvas |
| Pulsing live dots | Static dot with halo |
