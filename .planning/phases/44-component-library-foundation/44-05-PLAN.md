```typescript
  score: number;               // 0-100
  grade?: string;              // Auto-calculated if not provided
  size?: 'sm' | 'md' | 'lg';   // 64px, 96px, 128px
  showGrade?: boolean;
  className?: string;
}
```

SVG implementation (from gsd-phase0-component-library.md):
- Track circle: stroke="var(--surface-3)", stroke-width based on size
- Score arc: stroke="var(--accent)", stroke-linecap="round"
- Use pathLength={239} trick for easy dasharray calculation
- transform="rotate(-90 cx cy)" to start from top
- Center content: NumRow for score + italic grade text
- role="img" aria-label="Health score: {score}%"

Grade calculation: >= 90: A, >= 80: B+, >= 70: B, >= 60: C, < 60: D

Size config:
- sm: viewBox="0 0 64 64", r=24, stroke-width=5
- md: viewBox="0 0 96 96", r=38, stroke-width=7
- lg: viewBox="0 0 128 128", r=52, stroke-width=9
