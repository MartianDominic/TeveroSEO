## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, label margins |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, form group gaps |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level vertical spacing |

Exceptions:
- Touch targets minimum 44px for mobile (wizard navigation buttons, mode selection cards)
- Slider track height 4px with 20px thumb for precise voice blend control

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | oklch(1 0 0) / --color-background | Page background, card backgrounds |
| Secondary (30%) | oklch(0.967 0.001 286.375) / --color-secondary | Form inputs, inactive tabs, sidebar summary |
| Accent (10%) | oklch(0.546 0.245 262.881) / --color-primary | Primary buttons, active tab indicators, mode selection highlight, "Learn Voice" progress bar |
| Destructive | oklch(0.577 0.245 27.325) / --color-destructive | Delete protection rule, reset voice profile |

Accent reserved for:
- Primary CTA buttons ("Save Voice Profile", "Learn Voice", "Generate Preview")
- Active tab indicator in tabbed interface
- Selected mode card border in wizard
- Progress indicator during voice analysis
- Compliance score badges (passing threshold)

## Interaction States

### Voice Mode Cards

| State | Visual |
|-------|--------|
| Default | border-border, bg-card |
| Hover | border-primary/50, shadow-sm |
| Selected | border-primary (2px), ring-2 ring-primary/20, bg-primary/5 |
| Disabled | opacity-50, cursor-not-allowed |

### Voice Analysis Progress

| State | Visual |
|-------|--------|
| Idle | "Learn Voice" button enabled |
| Analyzing | Button shows spinner, progress bar appears below sidebar summary |
| Complete | Success badge, "Re-learn Voice" button |
| Failed | Error message with retry option |

### Compliance Score Colors

| Range | Color | Token |
|-------|-------|-------|
| >= 70 | Green | --color-success or custom oklch(0.6 0.2 145) |
| 50-69 | Amber | --color-warning or custom oklch(0.7 0.15 85) |
| < 50 | Red | --color-destructive |

Note: Add success/warning tokens to globals.css if not present.
