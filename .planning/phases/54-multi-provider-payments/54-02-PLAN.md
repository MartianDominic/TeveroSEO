  - Base URL switching: sandbox vs production
  - API version header: `Revolut-Api-Version: 2024-09-01`

- [ ] Implement `createPaymentSession(invoice)`
  ```typescript

  {
    amount: invoice.totalCents,
    currency: invoice.currency,
    description: `Invoice ${invoice.invoiceNumber}`,
  ```


### Signature Verification
```typescript
const payloadToSign = `v1.${timestamp}.${rawBody}`;
const expected = crypto.createHmac('sha256', secret).update(payloadToSign).digest('hex');
const isValid = crypto.timingSafeEqual(
  Buffer.from(signature.replace('v1=', '')),
  Buffer.from(expected)
);
```
