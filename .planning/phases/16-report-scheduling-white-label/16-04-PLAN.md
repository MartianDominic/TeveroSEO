## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-19 | Tampering | Footer HTML | mitigate | Sanitize HTML on save using DOMPurify in branding API (16-03); allow only safe tags (p, br, a, span) |
| T-16-20 | Information Disclosure | Logo file paths | mitigate | Serve logos through API route with auth, not direct filesystem access |
| T-16-21 | Denial of Service | Large logo in PDF | mitigate | Resize/compress logos on upload; max dimensions 200x60 |
| T-16-22 | Spoofing | XSS in footer | mitigate | Use DOMPurify to sanitize footer HTML on save in branding API |
</threat_model>
