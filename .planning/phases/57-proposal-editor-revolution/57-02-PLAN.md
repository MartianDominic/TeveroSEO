# Plan 57-02: Variable System + Resolution Service

## Goal
Implement the variable system with definitions, resolution, and palette UI.

## Variable Categories

| Category | Examples | Source | Color |
|----------|----------|--------|-------|
| client | `{{client.name}}`, `{{client.website}}` | Prospect record | Blue |
| provider | `{{provider.name}}`, `{{provider.vatNumber}}` | Workspace | Green |
| pricing | `{{totals.monthly}}`, `{{services.list}}` | Proposal services | Orange |
| audit | `{{audit.score}}`, `{{audit.issues}}` | Website audit | Purple |
| dates | `{{today}}`, `{{proposal.validUntil}}` | Computed | Gray |
| custom | `{{custom.projectCode}}` | Agency-defined | Teal |

## Tasks

### Task 1: Variable Schema (1.5h)

Create `variable_definitions` table:
- `id`, `workspaceId`, `templateId`
- `key` (unique), `label`, `labelEn`, `labelLt`
- `description` (localized)
- `category`, `sourceType`, `sourcePath`, `computation`
- `format`, `formatOptions`, `defaultValue`
- `isRequired`, `validationRules`
- `icon`, `displayOrder`

Seed system variables (~30 default variables).

### Task 2: VariableResolutionService (3h)

```typescript
async resolveVariables(proposalId: string, locale: 'en' | 'lt'): Promise<Record<string, string>>
```

- Load proposal, prospect, workspace, services, audit
- Resolve each variable based on sourceType:
  - `entity` → extract from sourcePath
  - `computed` → run computation function
  - `custom` → use stored value
  - `input` → prompt user
- Apply formatting (currency, date, number, etc.)
- Return map of `{{key}}` → resolved value

### Task 3: Variable Definition Service (1.5h)

CRUD for custom variables:
- `listAll(workspaceId)` — system + workspace variables
- `create(workspaceId, data)` — create custom
- `update(id, data)` — update custom
- `delete(id)` — delete custom (only workspace variables)

### Task 4: API Endpoints (1h)

```
GET    /api/variables                # List all (system + workspace)
GET    /api/variables/categories     # List categories
POST   /api/variables                # Create custom
PUT    /api/variables/:id            # Update custom
DELETE /api/variables/:id            # Delete custom
POST   /api/proposals/:id/resolve    # Resolve for proposal
```

### Task 5: VariablePalette Component (2h)

React component:
- Grouped by category with collapsible sections
- Search/filter functionality
- Drag handle for each variable
- Category color coding
- Tooltip with description
- Localized labels
