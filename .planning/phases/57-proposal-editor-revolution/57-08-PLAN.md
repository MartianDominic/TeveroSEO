# Plan 57-08: Clone + Undo/Redo + Magic Link

## Goal
Complete the editor with clone/duplicate, undo/redo, and magic link sharing.

## Dependencies

```bash
pnpm add zustand zundo
```

## Tasks

### Task 1: Zustand Store with Temporal (2h)

```typescript
import { create } from 'zustand';
import { temporal } from 'zundo';

const useProposalStore = create<ProposalState>()(
  temporal(
    (set) => ({
      content: initialContent,
      sectionOrder: initialOrder,

      updateSection: (sectionId, content) => set((state) => ({
        content: { ...state.content, [sectionId]: content },
      })),

      reorderSections: (newOrder) => set({ sectionOrder: newOrder }),
    }),
    { limit: 50 }
  )
);
```

### Task 2: Undo/Redo Keyboard Shortcuts (1h)

useEffect hook:
- Listen for Cmd+Z / Ctrl+Z → undo()
- Listen for Cmd+Shift+Z / Ctrl+Shift+Z → redo()
- Prevent default browser undo

UndoRedoButtons component:
- Undo button (disabled if !canUndo)
- Redo button (disabled if !canRedo)
- Toolbar placement

### Task 3: Duplicate Proposal (1.5h)

DuplicateButton component:
- Click triggers duplicate
- Modal: "Keep same prospect?" checkbox
- New name input (default: "Copy of ...")
- API: `POST /api/proposals/:id/duplicate`

Backend:
- Copy proposal record
- Copy all sections
- Copy services
- Reset status to draft
- New createdAt

### Task 4: Magic Link Generation (2h)

ShareModal component:
- Display generated link
- Copy button with toast
- Link expiry display (30 days)
- Regenerate button (invalidates previous)
- Share buttons: Email, WhatsApp, Copy

Backend:
- `POST /api/proposals/:id/link`
- Generate secure token
- Store token + expiry in proposal
- Return full URL

### Task 5: Public Proposal View (1.5h)

Route: `/p/[token]`
- Validate token
- Check expiry
- Render read-only proposal view
- Track view event
- Notify agency user
