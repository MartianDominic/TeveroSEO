# Plan 57-05: Custom Sections + Add Section Menu

## Goal
Enable adding custom sections of various types to proposals.

## Section Types

| Type | Description | Content Fields |
|------|-------------|----------------|
| text | Rich text block | `content: string` |
| image | Image with caption | `url, caption, alt` |
| testimonial | Quote with attribution | `quote, author, company, image?` |
| case_study | Mini case study | `title, metrics[], description` |
| video | Embedded video | `url, platform` |
| comparison | Before/after table | `items: {aspect, before, after}[]` |
| timeline | Project phases | `phases: {title, duration, description}[]` |
| custom | Generic block | `content: any` |

## Tasks

### Task 1: AddSectionMenu (2h)

Grid popup showing section types:
- 4x2 grid layout
- Icon + label for each type
- Hover preview/description
- Click creates section
- Localized labels

```
┌───────────┐ ┌───────────┐ ┌───────────┐ ┌───────────┐
│    📝     │ │    🖼️     │ │    💬     │ │    📊     │
│   Text    │ │   Image   │ │Testimonial│ │Case Study │
└───────────┘ └───────────┘ └───────────┘ └───────────┘
┌───────────┐ ┌───────────┐ ┌───────────┐ ┌───────────┐
│    🎬     │ │    ⚖️     │ │    📅     │ │    ➕     │
│   Video   │ │Comparison │ │ Timeline  │ │  Custom   │
└───────────┘ └───────────┘ └───────────┘ └───────────┘
```

### Task 2: Section Components (4h)

Create editor component for each section type:

**TextSection:** ProposalInlineEditor wrapper
**ImageSection:** Upload + URL input, caption field, alt text
**TestimonialSection:** Quote textarea, author, company, optional image
**CaseStudySection:** Title, metric cards (add/remove), description
**VideoSection:** URL input with platform detection (YouTube/Vimeo/Loom)
**ComparisonSection:** Before/after rows (add/remove)
**TimelineSection:** Phase cards (add/remove) with duration

### Task 3: Section CRUD API (1.5h)

```
POST   /api/proposals/:id/sections      # Add section
PUT    /api/proposals/:id/sections/:sid # Update section
DELETE /api/proposals/:id/sections/:sid # Delete section
```

- Position auto-calculated
- Section type validated
- Cascade delete with proposal

### Task 4: Delete Confirmation (0.5h)

AlertDialog for section deletion:
- "Are you sure?" message (localized)
- Cancel / Delete buttons
- Delete action removes from list and API
