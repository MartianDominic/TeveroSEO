# Plan 57-07: AI Content Generation

## Goal
Implement AI-powered content generation for proposal sections.

## Tasks

### Task 1: AIGenerationModal (2.5h)

Modal with:
- **Available Context** checkboxes:
  - Website Audit Results (score, issues)
  - Keyword Research (opportunities count)
  - Prospect Information (company, industry)
  - Competitor Analysis (if available)
- **Sections to Generate** checkboxes:
  - Hero / Introduction
  - Current State Analysis
  - Opportunities
  - ROI Projections
  - (Services and Investment use templates, not AI)
- **Tone & Style** dropdown:
  - Professional, consultative, ROI-focused
  - Friendly and approachable
  - Technical and detailed
  - Urgent and action-oriented
- **Language** radio: English / Lietuviu

### Task 2: ProposalAIGenerationService (4h)

```typescript
class ProposalAIGenerationService {
  async generateContent(
    proposalId: string,
    sections: string[],
    locale: 'en' | 'lt',
    tone: string
  ): Promise<GeneratedContent[]>
}
```

- Build context from proposal, prospect, audit, keywords
- Section-specific prompts (hero, current_state, opportunities, roi)
- Locale-aware prompts (EN/LT templates)
- Claude API integration
- Confidence scoring
- Variable suggestions extraction

### Task 3: Prompt Templates (1.5h)

Create prompts for each section:
- **hero**: Personalized opening hook
- **current_state**: Audit findings summary
- **opportunities**: Keyword opportunities
- **roi**: Traffic/revenue projections

Each prompt in EN and LT.

### Task 4: Generation Progress (1h)

AIGenerationProgress component:
- Section-by-section progress
- Streaming text display (if supported)
- Cancel button
- Error handling per section

### Task 5: API Endpoints (1h)

```
POST   /api/proposals/:id/generate        # Start generation
GET    /api/proposals/:id/generate/status # Poll status (or SSE)
```

- Queue generation job
- Return generated content
- Auto-create version with changeType: ai_generated

## Verification

# Select language: Lithuanian
# Click Generate
