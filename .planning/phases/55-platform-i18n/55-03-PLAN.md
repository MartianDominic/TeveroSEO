Create scripts/translate-messages.ts that:
- Takes input JSON path and output JSON path as arguments
- Loads existing output file if present (to skip already translated keys)
- Recursively traverses the input JSON object
- For each string value, calls TranslationService.translate() with context type 'ui' and formality 'formal'
- Preserves placeholders ({name}, {{count}}, %s, {0})
- Adds rate limiting delay between batches
- Writes output JSON with same structure

Add Lithuanian ICU plural forms (Lithuanian has 4 forms: one, few, many, other):
- one: 1, 21, 31, 41...
- few: 2-9, 22-29, 32-39...
- many: 11-19 (teens)
- other: 0, 10, 20, 30... and fractions

Example: "{count, plural, one {# potencialus klientas} few {# potencialus klientai} many {# potencialiu klientu} other {# potencialiu klientu}}"

Add npm scripts to package.json:
- "translate:web": "tsx scripts/translate-messages.ts apps/web/messages/en.json apps/web/messages/lt.json"
- "translate:seo": "tsx scripts/translate-messages.ts open-seo-main/src/i18n/locales/en.json open-seo-main/src/i18n/locales/lt.json"

Run translation for both apps (this will call Gemini API).
