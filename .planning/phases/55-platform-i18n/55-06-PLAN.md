Create open-seo-main/src/server/features/contracts/services/TemplateSubstitutionService.ts:

SubstitutionResult interface: { success, content, errors: string[], warnings: string[] }

TemplateSubstitutionService class:
- substituteVariables(sections, variables, values, targetLanguage, formality): Promise<SubstitutionResult>
  - Validate required variables present
  - Process sections in order
  - For each section:
    - Substitute {{placeholders}} with escaped values
    - If variable.translateValue && !section.isLegal && targetLanguage !== 'en': translate value
    - Check for unsubstituted variables, add to warnings
  - Return concatenated content
- escapeHtml(text): sanitize &, <, >, ", '
- escapeRegex(text): escape regex special chars
- formatDate(date, locale): locale-aware date formatting
- formatCurrency(amount, currency, locale): locale-aware currency formatting

Update ContractService:
- Import TemplateSubstitutionService and LanguageResolutionService
- generateAgreement(options): 
  - Resolve language via langService
  - Get template by language
  - Call substitutionService.substituteVariables
  - Store in generatedAgreements table
  - Return id, content, language, warnings
- createFromProposal(proposalId, prospectId, workspaceId):
  - Load proposal and workspace data
  - Build variableValues from proposal/workspace/prospect
  - Call generateAgreement
- getTemplate(templateId?, language): fetch from DB
- generateContractNumber(): create SEO-YYYY-NNNN format
