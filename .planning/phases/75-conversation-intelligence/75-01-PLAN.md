Create the ConstraintExtractor service that extracts structured AnalysisConstraints from unstructured client conversations using Claude Sonnet 4.6.

Purpose: This is the brain of the keyword intelligence system - everything downstream depends on accurate constraint extraction. The service converts natural language client conversations into typed constraints that drive geographic filtering, funnel classification, relevance scoring, and cascade selection.

- `types.ts` - Full AnalysisConstraints type system with all 7 constraint categories
- `prompts.ts` - XML-structured metaprompt with few-shot Lithuanian examples
- `ConstraintExtractor.ts` - Service class with extract() method, JSON parsing, confidence scoring
- `index.ts` - Module barrel export
</objective>
