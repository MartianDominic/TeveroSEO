# Plan 74-02: Multi-Model Embedding Generator

## Goal

Create a script that generates embeddings for all 3 Jina models with proper prefixes and caching.

## Implementation

Location: `AI-Writer/benchmark/embed_corpus.py`

### Model Configuration

```python
MODELS = {
    "v3": {
        "name": "jina-embeddings-v3",
        "query_prefix": "query: ",
        "doc_prefix": "passage: ",
        "api_task": None
    },
    "v5-nano": {
        "name": "jina-embeddings-v5-text-nano", 
        "query_prefix": "Query: ",
        "doc_prefix": "Document: ",
        "api_task": "retrieval"
    },
    "v5-small": {
        "name": "jina-embeddings-v5-text-small",
        "query_prefix": "Query: ",
        "doc_prefix": "Document: ",
        "api_task": "retrieval"
    }
}
```

### Functions

1. `embed_products(model_key, products, dimensions)` - Embed all product texts
2. `embed_queries(model_key, queries, dimensions)` - Embed all query texts  
3. `save_embeddings(embeddings, path)` - Cache to .npy files
4. `load_embeddings(path)` - Load cached embeddings

### Output

- `results/embeddings/{model}/products_768.npy`
- `results/embeddings/{model}/queries_768.npy`
- `results/embeddings/{model}/metadata.json` (latency, tokens)

## Verification

- All 3 models produce embeddings
- Dimension matches requested (768)
- Latency logged per model
