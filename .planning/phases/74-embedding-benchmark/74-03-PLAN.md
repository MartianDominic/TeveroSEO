# Plan 74-03: Retrieval Evaluation Metrics

## Goal

Implement standard IR evaluation metrics for comparing embedding quality.

## Implementation

Location: `AI-Writer/benchmark/evaluate.py`

### Metrics

1. **Recall@k** - % of relevant items in top-k results
```python
def recall_at_k(relevant: set, retrieved: list, k: int) -> float
```

2. **MRR@k** - Mean Reciprocal Rank (position of first relevant)
```python
def mrr_at_k(relevant: set, retrieved: list, k: int) -> float
```

3. **NDCG@k** - Normalized Discounted Cumulative Gain
```python
def ndcg_at_k(relevant: set, retrieved: list, k: int) -> float
```

4. **Category F1** - Per-category classification F1
```python
def category_f1(true_cats: list, pred_cats: list) -> dict
```
