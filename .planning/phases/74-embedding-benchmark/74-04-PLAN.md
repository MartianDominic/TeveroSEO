# Plan 74-04: Main Benchmark Orchestrator

## Goal

Create the main script that runs the full benchmark and generates a comparison report.

## Implementation

Location: `AI-Writer/benchmark/run_benchmark.py`

### Workflow

1. Load test data (products, queries)
2. For each model (v3, v5-nano, v5-small):
   - Generate embeddings (or load cached)
   - Run retrieval evaluation
   - Compute all metrics
3. Generate comparison table
4. Save results to JSON and Markdown

