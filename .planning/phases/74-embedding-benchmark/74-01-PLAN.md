# Plan 74-01: Lithuanian E-Commerce Test Corpus

## Goal

Create comprehensive Lithuanian e-commerce test data with ground truth labels for retrieval evaluation.

## Files to Create

### 1. lt_queries.json (50 queries)

Each query has:
- Lithuanian search query
- List of relevant product IDs (ground truth)
- Expected category
- Query type (navigational, transactional, informational)

### 2. mixed_queries.json (50 bilingual queries)

- 25 Lithuanian queries mapped to Lithuanian products
- 25 English queries mapped to same Lithuanian products (cross-lingual test)

## Implementation

Location: `AI-Writer/benchmark/data/`

```json
[
  {
    "id": "q001",
    "query": "šampūnas dažytiems plaukams",
    "query_en": "shampoo for dyed hair",
    "relevant_products": ["p001", "p002", "p003"],
    "category": "cat01",
    "type": "transactional"
  }
]
```

## Verification

- 50 queries cover all 20 categories
- Each query has 2-5 relevant products
- Bilingual pairs have matching semantics
