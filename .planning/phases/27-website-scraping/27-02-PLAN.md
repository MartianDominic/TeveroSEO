<objective>
Implement DataForSEO On-Page API fallback and hybrid scraping strategy for JS-rendered sites.

Purpose: Handle the ~20% of websites that use JavaScript rendering (React, Vue, Angular) where Cheerio fails. DataForSEO On-Page API renders pages with a headless browser at ~$0.02/page.

Output: Hybrid scraper that tries free Cheerio first, then falls back to paid DataForSEO On-Page API when needed, with consistent output format.
</objective>

       1. Normalize domain to https:// URL
