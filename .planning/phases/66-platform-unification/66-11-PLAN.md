test.describe('Connection Wizard', () => {

    // Step 1: Enter URL

    await page.click('[data-testid="continue-btn"]');

    await expect(page.getByText('Found:')).toBeVisible({ timeout: 10000 });

    await page.click('[data-testid="diy-option"]');

    await expect(page.getByText('Step 1 of')).toBeVisible();
    await page.click('[data-testid="next-step-btn"]');

    await page.click('[data-testid="copy-btn"]');
    await expect(page.getByText('Copied!')).toBeVisible();
    
    // Step 6: Verify (mock pixel ping)
    await page.click('[data-testid="verify-btn"]');
    // ... mock verification response
  });

  test('shows detection results for known platforms', async ({ page }) => {
    const platforms = [
      { url: 'https://example.myshopify.com', expected: 'Shopify' },
      { url: 'https://example.wordpress.com', expected: 'WordPress.com' },

# TeveroSEO Pixel Integration Guide

## Quick Start

Add this line to your website's `<head>`:

\`\`\`html
<script async src="https://pixel.tevero.io/t.js" data-site="YOUR_SITE_ID"></script>
\`\`\`

That's it! The pixel will start collecting analytics immediately.

## What the Pixel Collects

| Data Type | Description |
|-----------|-------------|
| Pageviews | Page URL, referrer, timestamp |
| Sessions | Unique visitor tracking |
| Core Web Vitals | LCP, CLS, INP metrics |
| Engagement | Scroll depth, click events |

## Features

### Real-time Analytics
View traffic and engagement metrics in your dashboard.

### Core Web Vitals
Monitor your site's performance with Google's CWV metrics.

### SEO Modifications (with approval)
Apply meta tag, schema, and content changes without touching your CMS.

## Advanced Configuration

\`\`\`html
<script async src="https://pixel.tevero.io/t.js" 
  data-site="YOUR_SITE_ID"
  data-no-cwv="true"  <!-- Disable CWV tracking -->
></script>
\`\`\`

## Platform-Specific Guides

- [WordPress Installation](./guides/wordpress.md)
- [Shopify Installation](./guides/shopify.md)
- [Wix Installation](./guides/wix.md)
- [Google Tag Manager](./guides/gtm.md)

## Privacy & Security

- No cookies required
- GDPR compliant
- Data encrypted in transit
- Processed in EU data centers

## Support

Questions? Contact support@tevero.io

# Platform Connection API

## Endpoints

### Detect Platform
\`\`\`
POST /api/connect/detect
Content-Type: application/json

{
  "url": "https://example.com"
}
\`\`\`

Response:
\`\`\`json
{
  "platform": "shopify",
  "confidence": 100,
  "features": ["ecommerce"],
  "paidPlanRequired": false,
  "estimatedTime": "2 min"
}
\`\`\`

### Get Installation Guide
\`\`\`
GET /api/connect/guide/:platform?siteId=xxx
\`\`\`

### Verify Installation
\`\`\`
GET /api/connect/verify?siteId=xxx
\`\`\`

Long-polls until pixel is detected (up to 30 seconds).

### Pixel Collection
\`\`\`
POST /api/pixel/collect
Content-Type: application/json

{
  "siteId": "xxx",
  "event": "pageview",
  "data": {
    "url": "/page",
    "referrer": "https://google.com"
  },
  "timestamp": 1234567890,
  "sessionId": "sess-123"
}
\`\`\`

### Get Pixel Analytics
\`\`\`
GET /api/pixel/:siteId/analytics?startDate=YYYY-MM-DD&endDate=YYYY-MM-DD
\`\`\`

## Error Codes

| Code | Description |
|------|-------------|
| SITE_NOT_FOUND | Invalid siteId |
| TOKEN_EXPIRED | Magic link expired |
| RATE_LIMITED | Too many requests |
