Create `DomChangeService`:

```typescript
export class DomChangeService {
  // Queue a new change for approval
  async queueChange(request: QueueChangeRequest): Promise<PixelDomChange>;
  
  // Approve a pending change (moves to 'approved' then 'live')
  async approveChange(changeId: string, userId: string): Promise<PixelDomChange>;
  
  // Reject a pending change
  async rejectChange(changeId: string, userId: string, reason?: string): Promise<void>;
  
  // Rollback a live change (creates reverse change)
  async rollbackChange(changeId: string, userId: string): Promise<PixelDomChange>;
  
  // Get approved changes for pixel to inject (filtered by URL)
  async getApprovedChanges(siteId: string, pageUrl?: string): Promise<ApprovedChangesResponse>;
  
  // Get pending changes for approval UI
  async getPendingChanges(siteId: string): Promise<PixelDomChange[]>;
  
  // Get all changes with pagination
  async getChangeHistory(siteId: string, options?: { limit?: number; offset?: number }): Promise<PixelDomChange[]>;
  
  // Capture current value before change (for diff)
  private async captureOldValue(installationId: string, changeType: string, targetUrl?: string): Promise<string | null>;
}
```

**Change lifecycle:**
1. `queueChange` -> status: 'pending'
2. `approveChange` -> status: 'approved' -> status: 'live' (immediate)
3. `rollbackChange` -> original status: 'rolled_back', new change: 'live'

**Pixel integration:**
- `getApprovedChanges` is called by pixel script to get current live changes
- Changes are filtered by targetUrl if provided
- Returns only necessary fields for DOM injection
