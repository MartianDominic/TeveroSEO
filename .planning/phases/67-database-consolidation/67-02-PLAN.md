## Task 2: Create Clients Migration Script

<read_first>
- open-seo-main/src/db/schema/shared-clients.ts
</read_first>

<action>
Create `scripts/db/migrate_clients.sql`:
```sql
BEGIN;

-- Log start
INSERT INTO _migration_log (migration_name, status) VALUES ('migrate_clients', 'started');

-- Create backup
CREATE TABLE IF NOT EXISTS _backup_open_seo_clients AS SELECT * FROM open_seo.clients;
CREATE TABLE IF NOT EXISTS _backup_alwrity_clients AS SELECT * FROM alwrity.clients;

-- Migrate from open_seo (primary source)
INSERT INTO shared_clients (id, workspace_id, ...)
SELECT id, COALESCE(workspace_id, 'ORPHAN_' || id), ...
FROM open_seo.clients
ON CONFLICT (id) DO NOTHING;

-- Merge from alwrity (non-overlapping)
INSERT INTO shared_clients (id, workspace_id, ...)
SELECT id, COALESCE(workspace_id, 'ORPHAN_' || id), ...
FROM alwrity.clients
WHERE id NOT IN (SELECT id FROM shared_clients)
ON CONFLICT (id) DO NOTHING;

-- Log completion
UPDATE _migration_log SET status = 'completed', completed_at = NOW() 
WHERE migration_name = 'migrate_clients' AND status = 'started';

COMMIT;
```
</action>

<acceptance_criteria>
- [ ] File exists: scripts/db/migrate_clients.sql
- [ ] Contains: `BEGIN;` and `COMMIT;`
- [ ] Contains: `ORPHAN_`
- [ ] Contains: `_migration_log`
</acceptance_criteria>
