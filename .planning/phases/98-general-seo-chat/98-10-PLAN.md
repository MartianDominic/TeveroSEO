# 98-10: World-Class Chat UX - Claude Code Patterns

## Vision

Transform SEO Chat to use **Claude Code CLI interaction patterns** - tool execution transparency, /commands, real-time streaming visibility, and minimal friction. Not web chat UX, but CLI-inspired directness.

## Claude Code Patterns to Implement

| Pattern | Web Chat | Claude Code | SEO Chat Implementation |
|---------|----------|-------------|------------------------|
| **Tool visibility** | Hidden behind spinner | Explicit "Running tool..." with args | Show tool cards with args + streaming results |
| **Commands** | Natural language only | /compact, /help, /clear | /analyze, /keywords, /proposal, /clear |
| **Stop** | Edit message, regenerate | Escape/Ctrl+C interrupt | Escape to stop, no message editing |
| **Input style** | Conversational | Direct command + args | Support both natural + `/command domain.com` |
| **Progress** | Generic spinner | Detailed tool output stream | Staged progress with tool args visible |
| **Results** | Integrated prose | Distinct tool result blocks | Clear tool result cards, not mixed in prose |

## Design System Alignment (v6 + v7)

| Principle | Application |
|-----------|------------|
| **One editorial moment** | Tool result IS the answer - no verbose wrapper |
| **Calm at rest, hover-to-reveal** | Copy/share actions hidden until hover |
| **Show what you did** | Tool execution log with args visible |
| **Show what you're doing** | Real-time streaming with partial results |
| **Trust formula** | Tool transparency × Stop capability × Predictable commands |

```tsx
// commands.ts
export const COMMANDS: Command[] = [
  {
    name: 'analyze',
    alias: ['a', 'domain'],
    description: 'Analyze domain health',
    usage: '/analyze <domain>',
    pattern: /^\/(?:analyze|a|domain)\s+(.+)$/i,
    execute: (domain) => `Analyze the domain ${domain}`,
  },
  {
    name: 'keywords',
    alias: ['k', 'kw'],
    description: 'Find keyword opportunities',
    usage: '/keywords <domain>',
    pattern: /^\/(?:keywords|k|kw)\s+(.+)$/i,
    execute: (domain) => `Find keyword opportunities for ${domain}`,
  },
  {
    name: 'feasibility',
    alias: ['f', 'check'],
    description: 'Check keyword feasibility',
    usage: '/feasibility <keyword>',
    pattern: /^\/(?:feasibility|f|check)\s+(.+)$/i,
    execute: (keyword) => `Check if we can rank for "${keyword}"`,
  },
  {
    name: 'proposal',
    alias: ['p', 'gen'],
    description: 'Generate proposal',
    usage: '/proposal',
    pattern: /^\/(?:proposal|p|gen)$/i,
    execute: () => `Generate a proposal for the keywords we've analyzed`,
  },
  {
    name: 'clear',
    alias: ['reset', 'new'],
    description: 'Clear conversation',
    usage: '/clear',
    pattern: /^\/(?:clear|reset|new)$/i,
    isLocal: true, // Doesn't send to AI
    execute: () => null,
  },
  {
    name: 'help',
    alias: ['h', '?'],
    description: 'Show available commands',
    usage: '/help',
    pattern: /^\/(?:help|h|\?)$/i,
    isLocal: true,
    execute: () => null,
  },
];

// Parse input for commands
export function parseCommand(input: string): ParsedCommand | null {
  for (const cmd of COMMANDS) {
    const match = input.match(cmd.pattern);
    if (match) {
      return {
        command: cmd,
        args: match.slice(1),
        text: cmd.execute?.(...match.slice(1)) ?? null,
      };
    }
  }
  return null;
}
```

### Task 9: Keyboard Shortcuts [HIGH]
**Files:** Create `apps/web/src/hooks/useKeyboardShortcuts.ts`

```tsx
const SHORTCUTS = {
  'mod+enter': { action: 'submit', description: 'Send message' },
  'escape': { action: 'stop', description: 'Stop generation' },
  'mod+k': { action: 'focusInput', description: 'Focus input' },
  'mod+shift+c': { action: 'clear', description: 'Clear conversation' },
  'mod+l': { action: 'copyLink', description: 'Copy proposal link' },
  '/': { action: 'startCommand', description: 'Start command' },
};

export function useKeyboardShortcuts(handlers: Record<string, () => void>) {
  useEffect(() => {
    const handleKeyDown = (e: KeyboardEvent) => {
      const mod = e.metaKey || e.ctrlKey;
      
      if (e.key === 'Enter' && mod && handlers.submit) {
        e.preventDefault();
        handlers.submit();
      }
      if (e.key === 'Escape' && handlers.stop) {
        e.preventDefault();
        handlers.stop();
      }
      if (e.key === 'k' && mod && handlers.focusInput) {
        e.preventDefault();
        handlers.focusInput();
      }
      if (e.key === 'c' && mod && e.shiftKey && handlers.clear) {
        e.preventDefault();
        handlers.clear();
      }
      // "/" starts command mode when not in input
      if (e.key === '/' && document.activeElement?.tagName !== 'TEXTAREA' && handlers.startCommand) {
        e.preventDefault();
        handlers.startCommand();
      }
    };
    
    window.addEventListener('keydown', handleKeyDown);
    return () => window.removeEventListener('keydown', handleKeyDown);
  }, [handlers]);
}
```
