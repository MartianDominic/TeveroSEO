| --space-1 | 4px | Icon gaps, inline padding |

## Interaction Patterns

### Parallel Streaming UX (D-01)

1. **Instant (<200ms)**: Right Rail shows skeleton cards with pulse animation
2. **0-2s**: Chat streams conversational analysis text
3. **As data arrives**: Right Rail cards fill with micro-animations:
   - Numbers count up (500ms ease-out)
   - Card slides in (280ms --ease-smooth)
   - Subtle glow pulse when updated (160ms --motion-fast)
4. **Highlight sync**: When prose mentions metric, corresponding card briefly highlights (border-accent for 160ms)

### Error Recovery (D-02)

- 3 automatic retries with exponential backoff (1s, 2s, 4s)
- Inline "Retry" button appears after 3 failures
- Succeeded tool results preserved during partial retries
- Error message tone: agency-smart, explains WHY, offers action

### Topical Map Interaction (D-03)

| Level | Agency Side | Prospect Portal |
|-------|-------------|-----------------|
| View-only | Pan/zoom only | Pan/zoom only (default) |
| Interactive | Click, drag, expand/collapse, minimap | Enabled per-prospect toggle |

### Animation Tokens

| Animation | Duration | Easing | Usage |
|-----------|----------|--------|-------|
| Card hover lift | 280ms | var(--ease-smooth) | All cards on hover |
| Count-up | 500ms | ease-out | KPI numbers filling |
| Skeleton pulse | 1.5s infinite | var(--ease-smooth) | Loading states |
| Slide-in | 280ms | var(--ease-smooth) | Right Rail cards appearing |
| Highlight sync | 160ms | var(--ease-quick) | Prose-to-card connection |
| Message appear | 240ms | var(--ease-smooth) | New chat messages |
