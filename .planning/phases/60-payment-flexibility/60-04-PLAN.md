Create DiscountCodeService with:
- validateCode(workspaceId, code, amountCents) per D-13 to D-16:

  - Check validFrom <= now <= validUntil
  - Check usedCount < maxUses (if maxUses set)

  - percentage: Math.floor(amountCents * (1 - discountValue / 100))
  - fixed: Math.max(0, amountCents - discountValue)
  - Returns discountedAmountCents

- applyCode(codeId):
  - Increments usedCount
  - Returns updated code

Validation error messages:
- "Invalid discount code"
- "This code has expired"
- "This code has reached its usage limit"
